# 🧠 DevQuiz — Software Engineering Quiz App

A browser-based interactive quiz application covering core **Software Engineering concepts**. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

---

## ✨ Features

- ⏱ **Countdown Timer** — Each question is timed to keep you on your toes
- 🎯 **Score Tracking** — Your score is calculated and displayed at the end
- 🔀 **Randomized Questions** — Questions shuffle every session for a fresh experience
- 📊 **Multiple Difficulty Levels** — Progress from beginner to advanced topics
- 📱 **Responsive Design** — Works on desktop and mobile browsers

---

## 📚 Topics Covered

Questions span key Software Engineering areas including:

- Software Development Life Cycle (SDLC)
- Data Structures & Algorithms
- Object-Oriented Programming (OOP)
- Software Quality & Testing
- Databases & SQL
- Networking & Security basics
- Design Patterns

## 🚀 Getting Started

No installation needed. Just open in a browser.

```bash
# Clone the repository
git clone https://github.com/tasimbiswakandemiiri919-arch/quiz-app.git

# Navigate into the folder
cd quiz-app

# Open in browser
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```
